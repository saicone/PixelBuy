## Requisitos

* Al menos Minecraft 1.8.8.
* Como mínimo Java 11.

## Descarga

No hay link de descarga.

## Compatibilidad

| Servidor | Compatible |
|----------|------------|
| Bukkit   |     ✅    |
| Spigot   |     ✅    |
| Paper    |     ✅    |
| Folia    |     ❌    |

## Instalación

1. Entra al explorador de archivos de tu servidor.
2. Abre la carpeta `plugins`.
3. Sube el archivo `.jar` de PixelBuy.
